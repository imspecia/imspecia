Initially I struggled with this interaction because of how simple or how difficult I could interpret it. I sketched down initial ideas of creating a day and night kind of concept where I would have a button or scroll to change the time in day. This concept tumbled into other ideas that have to deal with growth or for example: how long it takes for a candle to melt. Most of my initial ideas were straightforward with the concept of making a lot of games or interactions to use up your time. I started researching elements I wanted to incorporate into my website like: having a timer that automatically times how long you are on the browser, dark mode (for night time), and a countdown timer for a day of your choice.

Mood Board Inspiration:

I started off with coding a countdown timer. [inspiration:]
I started off using this code to get the idea of how to make a countdown timer but you have to manually change the month and day in the javascript. After playing around with code, I created a submission box where you would have to type out a day/ month/ year that you wanted to countdown from but it ended up with a lot of errors if it was not inputted correctly. The solution I found was using a calendar system so you can just pick any date in the future. I had to change minor issues like restricting the user from inputting a date that is before the present day. The background was initially a solid colour but then I opted to create a gradient background with the colour palette I used. I created a separate html and css file to create circles that spell out “ELEMENTS” with coloured bordered circles and solid coloured circles animating and spelling out each letter. I created this animation to add a visual to the website. When you first open this website it might take you a minute or so to realize what the circles spell out. As the circles move, I see myself following each moving circle as it loops endlessly, tracing each letter form.

Experimenting:

I looked into how to make a browser startup timer as well as a dark mode toggle. Down below you can see the two different variations with the inital background colours. The brown was the startup colour and the black was the dark mode colour. I created a colourful timer when it is 'light mode' and a simple white colour timer when it is in 'dark mode'. When you hover over the "Toggle Dark Mode" button, it will change colours. I decided to combine my countdown timer within the same page instead of creating a button to make an external page. Using containers, I lined up all my elements and created an illusion of having a second page at the bottom.

FINAL:

Here are the final colours. I decided that brown and black were too similar and there was not much contrast so I experimented with what background colour works well with the animation. This stategrey colour creates a more subtle look to "ELEMENTS" but it does not take away from it. I chose to keep the brown colour and have it as the 'dark mode' colour instead to make the colours pop more.
