LOADER:

I added a loader html that redirects you to the actual page after a few seconds of waiting. Using a simple animation and my colour palette, I was able to create a preloading screen for my website.

Reading Responses:

[Resource:]
As I was creating my index page, the reading responses was a bit tricky to do. I decided to use the code from w3schools that creates vertical tabs. In order to keep the words hidden, I had to modify the code so that you can see the titles of the words when you hover over the tabs. I thought this method would be the easiest and best way to display all the reading responses in one place.

FINAL:

Here are the final colours. I decided that brown and black were too similar and there was not much contrast so I experimented with what background colour works well with the animation. This stategrey colour creates a more subtle look to "ELEMENTS" but it does not take away from it. I chose to keep the brown colour and have it as the 'dark mode' colour instead to make the colours pop more.
